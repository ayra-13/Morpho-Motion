# 3D Animated Website

This project is a simple 3D animated website using Three.js. It displays a rotating 3D cube.

## Live Demo
[Click Here](https://github.com/ayra-13/Morpho-Motion/3D-Cube)

## Features
- 3D Cube Animation
- Responsive Canvas
- Interactive with Three.js

## Installation & Usage
### **Option 1: Open Locally**
1. Download the `index.html` file.
2. Open it in a web browser (Chrome recommended).

### **Option 2: Clone from GitHub**
```sh
git clone https://github.com/ayra-13/Morpho-Motion
cd 3D-Website
```
Then, open `index.html` in a browser.

## Requirements
No additional dependencies needed. Just a modern browser!
